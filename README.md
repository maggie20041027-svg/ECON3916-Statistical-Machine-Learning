## About this Portfolio

This repository hosts my ECON 3916 coursework and labs, built around a **Concept Extension** philosophy: we begin with foundational statistical concepts (e.g., **regression**) and extend them to scalable machine learning methods (e.g., **Lasso** and other regularization techniques).

Through these projects, I am learning to combine:

- **Causal inference** (economic theory + econometrics mindset)
- **Predictive analytics** (machine learning + out-of-sample performance)

The result is a toolkit that supports both **interpretability** (why something happens) and **prediction** (what is likely to happen next).
